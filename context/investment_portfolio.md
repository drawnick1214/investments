## 🧮 Interest Accrual Formulas

### Daily interest from EA rate:

```
Daily rate = (1 + EA)^(1/365) - 1

Savings (8.75% EA):    Daily = (1.0875)^(1/365) - 1 = ~0.02298% per day
CDT #1  (9.00% EA):    Daily = (1.09)^(1/365) - 1   = ~0.02358% per day
CDT #2  (9.75% EA):    Daily = (1.0975)^(1/365) - 1  = ~0.02554% per day
```

- **Frontend:** React (web app or PWA)
- **Storage:** LocalStorage or simple JSON file per day

1. **Dashboard** — total portfolio value in USD and COP
2. **Daily snapshot form** — enter closing prices for each asset

6. **Currency converter** — input today's TRM to convert COP→USD
